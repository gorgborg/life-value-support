# Life Value – App Support

**Life Value** is an iOS application designed to help users create and maintain a living record of their family across generations.

If you need help using the app, have questions, or wish to report an issue, please use the contact details below.
---
### Support

### About the App
Life Value allows users to organise family information, track generations, and preserve meaningful details in a clear and structured way. The app is designed with privacy and simplicity in mind.
---
### Privacy
Life Value does not require users to create accounts.
Any data entered into the app remains under the user’s control.

For privacy-related enquiries, please contact the email address above.
